# Q2: Basic Routing

Inside the template that is given, create the following routes:

## /
Returns the string "hi everybody!"

## /about
Returns the string "i am very smart"

## /double/\<number>
Returns the double (i.e, multiply by 2). 

Hints: 
* you can only return an array, a dictionary or a string from a Flask route
* All route arguments are passed in as strings

## /add/\<n1>/\<n2>
Return the sum of n1 and n2
